# CSS Advanced Project

This project focuses on implementing the advanced styling of a webpage using **CSS**, following a Figma design specification.

## 📌 Project Overview
The goal of this project is to:
- Translate a provided **Figma design** into a responsive and pixel-perfect webpage.
- Apply advanced CSS properties for layout, typography, and colors.
- Maintain semantic HTML structure from the **HTML, advanced** project.
